# Proyecto de Análisis de Datos de Vehículos

Este proyecto tiene como objetivo poner en práctica tareas habituales de ingeniería de software y análisis de datos. Se ha trabajado un conjunto de datos de anuncios de venta de coches (en este caso, se utiliza `vehicles_us.csv` como dataset) y se ha desarrollado una aplicación web con Streamlit que permite visualizar un histograma y un gráfico de dispersión presentados después de activar un par de botones que los invocan.

## Funcionalidades de la aplicación web

- Encabezado con descripción del proyecto.
- Botón para generar un histograma del año del modelo de los vehículos
- Botón para generar un gráfico de dispersión relacionando el precio con el kilometraje.
